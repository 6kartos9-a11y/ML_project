# 📊 Salary Prediction using Linear Regression

## 📌 Overview

This project demonstrates how Linear Regression, a supervised machine
learning algorithm, can be used to predict employee salaries based on
their years of experience.

## 🎯 Objective

-   Build a regression model to predict salary\
-   Understand relationship between experience and salary\
-   Evaluate model performance

## 🗂️ Dataset

-   YearsExperience: Input feature\
-   Salary: Target variable

## ⚙️ Technologies Used

-   Python\
-   Pandas\
-   NumPy\
-   Matplotlib\
-   Seaborn\
-   Scikit-learn

## 🔍 Steps

1.  Data preprocessing\
2.  Exploratory Data Analysis\
3.  Model training using Linear Regression\
4.  Prediction\
5.  Evaluation using R² score

## 📈 Results

-   Strong linear relationship observed\
-   Good model performance

## ⚠️ Limitations

-   Only one feature used\
-   Not fully realistic

## 🚀 Future Improvements

-   Add more features\
-   Use advanced ML models

## ▶️ How to Run

``` bash
git clone <repo-link>
pip install -r requirements.txt
jupyter notebook
```

## 📌 Conclusion

A simple and effective implementation of linear regression for salary
prediction.
